# 垃圾車即時追蹤與警報系統 (Garbage Truck Tracker & Alert)

一個現代化、高質感的垃圾車即時追蹤與雙點定位警報 App。支援地圖自訂住家與警報偵測點、特定時間過濾、路段關鍵字篩選、瀏覽器推播通知及語音廣播功能。

## 🚀 Vercel 完全免費託管部署指南 (無須填寫信用卡)

此專案已內建 `vercel.json` 與無伺服器 API 架構，您可以非常快速地在 Vercel 上進行免費部署：

### 第一步：註冊 Vercel
1. 前往 [Vercel 官網](https://vercel.com/)。
2. 點選 **「Sign Up」**，直接選擇使用 **GitHub 帳號註冊與登入**（完全不需要綁定任何信用卡）。

### 第二步：導入專案部署
1. 進入 Vercel 後，點選右上角或首頁的 **「Add New...」** -> **「Project」**。
2. 在 GitHub 列表中，點擊您剛才上傳的專案（`g9502995/-web`）右側的 **「Import」**。
3. 進入設定畫面，直接維持預設選項不用修改。
4. 點擊最下方的 **「Deploy」** 按鈕。
5. 等待大約 30 秒至 1 分鐘，看到滿天灑花圖案即表示部署成功！

部署完成後，Vercel 會提供您一個免費的專屬網址（例如：`https://your-app-name.vercel.app`），即可在手機或電腦上直接打開並開始使用！

---

## 💻 本地執行方式
如果您想在自己的電腦測試：
1. 確保電腦已安裝 Node.js。
2. 於專案目錄執行：
   ```bash
   npm install
   npm start
   ```
3. 開啟瀏覽器造訪 `http://localhost:3000`。
